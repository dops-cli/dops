# Installation

<!-- tabs:start -->

#### ** Windows **

```powershell
iwr -useb dops-cli.com/get/windows | iex
```

#### ** Linux **

```bash
curl -s https://dops-cli.com/get/linux | sudo bash
```

#### ** macOS **

```bash
curl -s https://dops-cli.com/get/linux | sudo bash
```

#### ** Compile from source **

```sh
go get github.com/dops-cli/dops
```
